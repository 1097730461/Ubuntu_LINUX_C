一、功能设计：

    1.将摄像头拍摄到的画面在LCD显示；

    2.I.MX监测Pbot电机的状态并在UI显示；
        （通过CAN协议向I.MX发送信息，通过文件读写操作最终在UI上显示）

    3.PC/手机，远程监控Pbot状态，并且可以远程控制Pbot；
        （物联网）（socket）
        （PC的ubuntu/I.MX——客户端/服务器）
